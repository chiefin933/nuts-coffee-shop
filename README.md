Local development

1. Serve the site locally using a static server. Example with `http-server`:

```bash
# install (once)
npm install --global http-server
# run in the project directory
http-server -p 8080
# open http://localhost:8080
```

2. Edit files in your editor and refresh the browser to see changes.

Deploying

This repository includes a GitHub Actions workflow that automatically deploys the site to GitHub Pages whenever you push to `master`.
